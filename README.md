# AI-SEOS

AI-powered SEO optimization system with modular architecture.

## Project Structure

```
AI-SEOS/
├── packages/          # Core packages
│   ├── core/         # Core functionality
│   ├── shared/       # Shared utilities and types
│   ├── scanner/      # SEO scanning engine
│   ├── evidence/     # Evidence collection and analysis
│   ├── memory/       # Memory management and caching
│   ├── runtime/      # Runtime execution engine
│   └── reports/      # Report generation
├── apps/             # Applications
│   ├── studio/       # Web studio interface
│   └── api/          # Backend API
├── docs/             # Documentation
└── README.md
```

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

## Packages

### Core (`packages/core`)
Main core functionality for AI-SEOS.

### Shared (`packages/shared`)
Shared utilities, types, and constants.

### Scanner (`packages/scanner`)
SEO scanning engine for website analysis.

### Evidence (`packages/evidence`)
Evidence collection and analysis system.

### Memory (`packages/memory`)
Memory management and caching layer.

### Runtime (`packages/runtime`)
Runtime execution engine.

### Reports (`packages/reports`)
Report generation and formatting.

## Applications

### Studio (`apps/studio`)
Web-based interface for AI-SEOS.

### API (`apps/api`)
Backend API server.
